MultiPlayer Snake

My take on an arcade classic that we all know and love. Used to showcase a realtime data application for PubNub's platform technology. Note: Open multiple tabs to test out the online multiplayer.

Best Scrabble Word Finder

An instant-search word finder that helps you maximize your Scrabble points. This has been a fun project for me as I have rebuilt it using different technology stacks over time to both learn and play with efficiency. The current iteration is entirely front-end processed with JavaScipt.

Live Version GitHub

Reciprocity Map

A friend of mine is a concealed weapons instructor. He wanted a simple resource on his website to educate his students on where it is legal to conceal carry. I built him this interactive map to fulfill that need. The application uses HTML, CSS, JavaScript, PHP, and MySQL.

Live Version GitHub

eMail Scanner

This project is purely a back-end service that scans email addresses at an interval, pulls and cleans up meta-data, and dumps voicemail attachments into a network shared drive. This was my first dive into Node.js, so it was interesting working with JavaScript from the back-end perspective.
